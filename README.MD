# Positron — Urban Delivery Simulator

15×15 grid-based urban delivery simulation. A robot navigates sequential delivery
destinations using BFS, DFS, UCS, Greedy Best-First, and A*. Features a live 2D
grid viewer, algorithm benchmarking dashboard, and a 3D interactive search-tree
visualiser inspired by Palantir Foundry.

**Stack:** Python 3.11+ · React 18 · Three.js · Zustand · Recharts · Tailwind

## Quickstart

```bash
# Backend
cd backend && python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt && python main.py

# Frontend
cd frontend && pnpm install && pnpm dev
```
